# Datasheet / reference links

Raw datasheet PDFs are intentionally not vendored in this public repo. The hardware notes and schematic audit were checked against these public reference families:

- Espressif ESP32 DevKit / ESP32-WROOM-32 datasheet and hardware design guidelines.
- Texas Instruments SN74AHCT125 / SN74AHCT125N quad bus buffer datasheet.
- Worldsemi / WS2811 12 V LED pixel-strip protocol references.
- Mean Well RS-150 series power-supply specification.
- BusBoard Prototype Systems BB830 breadboard datasheet.
- Vishay / equivalent 1N5820-1N5822 Schottky diode datasheets.
- Expert Sleepers ES-9 user manual and hosted-mode configuration notes.
- 4ms Spherical Wavetable Navigator manual.
- Ornament & Crime / Hemisphere / Phazerville Logic applet documentation.

When rebuilding the hardware pack, download the current vendor PDF revisions locally and keep them out of git unless license terms explicitly allow redistribution.
